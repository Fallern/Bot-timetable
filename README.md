# Bot-timetable

## Using

Clone repo:

```bash
git clone https://github.com/Fallern/Bot-timetable.git.git && cd Bot-timetable
```

Create venv:

```bash
python3 -m venv venv
```

Activate venv:

```bash
source venv/bin/activate
```

Then install packages from requirements.txt:

```python
pip install -r requirements.txt
```
